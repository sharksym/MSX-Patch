란마 1/2의 메모리 매퍼 관리 루틴을 변경하는 패치입니다.

메인 메모리 매퍼가 4MB일 때 발생하는 메모리 문제가 해결됩니다.

패치방법: 아래 IPS 파일을 1번 디스크 이미지에 적용합니다.
FILE: RANMA_1-2_(1-8)_MemoryMapper.ips

참고)
오리지널 및 한글판 란마 1번 디스크에 모두 적용할 수 있습니다.
패치 적용 후 디스크 이미지를 읽기전용(쓰기방지)으로 설정하세요.
